# EggWars Plugin for PocketMine-MP

بلوجين EggWars كامل لماين كرافت البيدروك باستخدام PocketMine-MP API 5

## التثبيت

1. ضع مجلد البلوجين في `plugins/EggWars/` داخل سيرفر PocketMine-MP
2. أعد تشغيل السيرفر
3. اضبط الساحات في `plugins/EggWars/arenas.yml`

## الميزات

- **نظام الساحات المتعددة**: دعم عدة ساحات في نفس الوقت
- **إدارة الفرق**: 4 فرق ملونة (أحمر، أزرق، أخضر، أصفر)
- **حماية البيض**: الهدف الرئيسي للعبة
- **مولدات الموارد**: إنتاج تلقائي للحديد والذهب والماس
- **نظام المتجر**: شراء الأسلحة والدروع والكتل
- **الإحصائيات**: تتبع القتل والوفيات لكل لاعب

## الأوامر

- `/eggwars join <arena>` - الانضمام لساحة
- `/eggwars leave` - مغادرة الساحة الحالية
- `/eggwars list` - عرض جميع الساحات المتاحة
- `/eggwars shop` - فتح المتجر (أثناء اللعب فقط)
- `/eggwars stats` - عرض الإحصائيات الشخصية

### أوامر الإدارة

- `/eggwars reload` - إعادة تحميل الإعدادات
- `/eggwars create <arena>` - إنشاء ساحة جديدة (قريباً)

## الإعداد

### إعداد الساحات

عدّل ملف `arenas.yml` لإضافة ساحات جديدة:

```yaml
my_arena:
  world: "eggwars_world"
  min_players: 4
  max_players: 16
  lobby_spawn:
    x: 0
    y: 100
    z: 0
  teams:
    red:
      color: "§c"
      spawn:
        x: 50
        y: 100
        z: 0
      egg:
        x: 45
        y: 101
        z: 0
```

### إعداد المتجر

عدّل ملف `shop.yml` لتخصيص عناصر المتجر والأسعار.

- PocketMine-MP 5.0.0+
- PHP 8.0+
- عوالم ماين كرافت محضرة للساحات
